# 缩放拖拽

- canvas可以理解为一张无限大的纸，页面上有一个固定不动的窗口放画布，拖拽或者缩放的时候窗口不动，只是画布动
- offset记录画布整体平移量，一开始是（0，0）
- （0，0）是画布区域的左上角，这是图形学的规定

# 添加元素

### 图形

#### 1. 工具状态设置（在 toolbars.js 中）

形状面板中点击矩形方框时， toolbars.js 中的 initShapeSelection 函数会被触发，将 window.currentTool 的值设置为 "rect" 。这样就标记了当前的工具状态为矩形绘制模式。

#### 2. 鼠标事件监听（在 graph.js 中）

graph.js 文件中的 initGraphDrawing 函数会在 canvas 元素上添加鼠标事件监听器，具体流程如下：

###### 鼠标按下事件（mousedown）

- 当你在画布上按下鼠标时，监听器会检查 window.currentTool 的值
- 如果 window.currentTool === "rect" ，就调用 startDrawingRect 函数开始绘制
- startDrawingRect 函数会：
  - 创建一个矩形元素（使用 createElement("rect") ）
  - 记录鼠标按下的起点坐标
  - 将矩形元素添加到 elements 数组中
  - 调用 render 函数重绘画布，显示初始的矩形（此时宽度和高度为 0）

###### 鼠标移动事件（mousemove）

- 当你按住鼠标并在画布上移动时，监听器会检查 window.currentTool === "rect" 且 isDrawing 为 true
- 如果条件满足，就调用 drawRect 函数
- drawRect 函数会：
  - 计算鼠标当前位置与起点的距离，得到矩形的宽度和高度
  - 更新当前矩形元素的 width 和 height 属性
  - 调用 render 函数重绘画布，实时显示矩形的大小变化

###### 鼠标释放事件（mouseup）和鼠标离开事件（mouseleave）

- 当你松开鼠标或鼠标离开画布时，监听器会将 isDrawing 设置为 false ，结束绘制过程
